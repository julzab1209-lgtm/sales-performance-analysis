# Visualizations

## Overview

The Sales Performance Dashboard uses multiple visualization types to present sales data in a clear, interactive, and business-oriented way.

Each visualization was selected to answer specific analytical questions and support decision-making.

---

## Monthly Sales Trend

**Visualization Type**

Line Chart

**Purpose**

The line chart presents changes in sales performance over time.

**Business Question**

- How have sales changed month by month?
- Are there any seasonal trends?
- Which periods generated the highest sales?

---

## Sales by Product

**Visualization Type**

Bar Chart

**Purpose**

The bar chart compares sales performance across different products.

**Business Question**

- Which products generate the highest revenue?
- Which products underperform?
- How does product performance compare?

---

## Sales by Country

**Visualization Type**

Pie Chart

**Purpose**

The pie chart illustrates each country's contribution to total sales.

**Business Question**

- Which countries contribute the most to overall sales?
- How is sales distributed across markets?

---

## Sales Map

**Visualization Type**

Map Visualization

**Purpose**

The map displays the geographic distribution of sales.

**Business Question**

- Where are the strongest markets located?
- Which regions generate the highest sales?
- Are there geographical sales patterns?

---

## Country Filter

**Visualization Type**

Slicer

**Purpose**

The slicer enables users to filter the dashboard dynamically by country.

**Business Value**

- Interactive analysis
- Regional comparisons
- Faster exploration of sales data

---

## Interactive Features

The dashboard includes several interactive capabilities:

- Cross-filtering between visuals
- Dynamic filtering
- Drill-down analysis
- Interactive selections
- Real-time visual updates

These features improve usability and support efficient business analysis.

---

## Summary

The selected visualizations provide a comprehensive overview of sales performance while allowing users to explore the data from multiple perspectives. Together, they support informed, data-driven business decisions.
