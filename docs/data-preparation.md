# Data Preparation

## Overview

The dataset was prepared in Microsoft Power BI to ensure consistency and support accurate analysis.

---

## Data Source

The dataset includes information about:

- Products
- Countries
- Sales
- Units Sold
- Dates
- Discounts

---

## Data Preparation

Data was prepared using **Power Query**, including:

- Data type validation
- Column formatting
- Data transformation
- Table organization

---

## Data Model

A structured data model was created to support:

- Product analysis
- Country analysis
- Time-based analysis
- Interactive filtering

---

## Outcome

The prepared dataset provides a reliable foundation for interactive reporting and business analysis in Power BI.
