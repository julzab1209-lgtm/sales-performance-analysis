# Dashboard Overview

## Overview

The Sales Performance Dashboard provides an interactive view of sales performance across products, countries, and time periods. It enables users to explore sales trends and compare performance using interactive visualizations.

---

## Dashboard Components

### Monthly Sales Trend

Line chart showing monthly sales performance and trends.

### Sales by Product

Bar chart comparing sales across products.

### Sales by Country

Pie chart presenting the distribution of sales by country.

### Sales Map

Interactive map visualizing sales across different countries.

### Country Filter

Slicer for dynamically filtering all dashboard visualizations.

---

## Interactive Features

The dashboard supports:

- Cross-filtering
- Drill-down analysis
- Dynamic filtering
- Interactive selections

---

## Purpose

The dashboard helps users monitor sales performance, identify trends, and support data-driven business decisions.
